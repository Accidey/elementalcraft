ElementalCraft —— 元素属性战斗模组

ElementalCraft 是一个为 Minecraft 添加全新战斗机制的模组。它通过元素属性系统，让武器和护甲拥有赤焰、冰霜、雷霆、自然等元素力量，从而带来更丰富的战斗体验。

核心功能
元素攻击：武器可附加元素攻击属性，对敌人造成额外伤害，并触发克制关系（例如赤焰克制冰霜）。

元素强化与抗性：护甲可获得元素强化和抗性，提升防御或抵消特定元素伤害。

维度强制属性：

下界生物自动获得火焰属性。

末地生物自动获得雷霆属性。

如果装备槽不足，会自动补齐铁质护甲并赋予最高等级的元素属性。

强制物品配置：可在配置文件中指定某些物品必须带有特定元素属性，确保玩法一致性。

随机生成机制：普通怪物有一定概率随机生成元素属性，增加战斗的多样性。

调试模式：开发者和玩家可开启调试模式，实时查看战斗中的元素伤害计算过程。

模组亮点
完整的元素体系：赤焰、冰霜、雷霆、自然，互相克制，形成策略性战斗。

全覆盖的装备逻辑：无论是合成、熔炼、锻造还是怪物掉落，装备都会自动应用元素属性。

高度可配置：通过配置文件可自由调整属性数值、掉落概率、维度规则等。

开源友好：代码注释完整，中英文双语，方便二次开发与社区交流。

English Description
ElementalCraft — Elemental Attribute Combat Mod

ElementalCraft introduces a brand-new combat system to Minecraft by adding elemental attributes to weapons and armor. With fire, frost, thunder, and nature powers, battles become more strategic and immersive.

Core Features
Elemental Attacks: Weapons can gain elemental attack attributes, dealing extra damage and triggering elemental restraint rules (e.g., fire restrains frost).

Enhancement & Resistance: Armor can be enhanced with elemental strength or resistance, boosting defense or reducing specific elemental damage.

Dimension-Forced Attributes:

Nether mobs automatically gain fire attributes.

End mobs automatically gain thunder attributes.

If armor slots are missing, iron armor will be generated and infused with maximum-level elemental attributes.

Forced Item Config: Define specific items in the config to always carry certain elemental attributes, ensuring consistent gameplay.

Random Generation: Regular monsters have a chance to spawn with random elemental attributes, adding variety to encounters.

Debug Mode: Developers and players can enable debug mode to view detailed elemental damage calculations during combat.

Highlights
Complete Element System: Fire, Frost, Thunder, Nature — each with strengths and weaknesses, creating tactical combat depth.

Universal Equipment Logic: Covers crafting, smelting, smithing, and mob drops — items always receive elemental attributes.

Highly Configurable: Adjust attribute values, loot chances, dimension rules, and more via config files.

Open-Source Friendly: Fully documented with bilingual comments (Chinese & English), making it easy for community contributions and further development.
